# BulkMail Pilot ✈️

A responsive, full-stack web application for automated bulk email sending with attachments.

## Features
- **Dashboard**: Real-time sending statistics and logs.
- **Compose**: HTML email support, attachments, and `{Name}` personalization.
- **Recipients**: Import from CSV/Excel or manage manually.
- **Templates**: Save and load reusable email templates.
- **Tech Stack**: Python (Flask), SQLite, Vanilla JS, CSS (Dark Mode).
